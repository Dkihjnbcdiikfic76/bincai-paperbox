The Bincai Color Management Stack

1. Press-Level Spectrophotometry (Heidelberg Speedmaster XL 105-7+UV)

Our Heidelberg Speedmaster XL 105 7-color + UV coater is equipped with Heidelberg Prinect Inpress Control 3 — an inline spectrophotometer that measures every single sheet as it comes off the press. Unlike offline sampling (where you measure 1 sheet every 500–1000), Inpress Control scans the entire print run in real time.

What this means for your brand:

  • Real-time L*a*b* color measurement at 18,000 sheets/hour
  • Automatic ink-key adjustment when color drifts beyond ΔE 1.5
  • No batch-to-batch variation — every sheet is a “checked sheet”
  • PDF report generation for full color traceability

2. Densitometric Control (KBA Rapida 105 4-Color)

Our KBA Rapida 105 4-color offset press uses Techkon SpectroDens handheld spectrophotometers for offline density checks every 500 sheets, supplemented by the press’s integrated ErgoTronic ColorControl system for inline density monitoring.

3. ISO 12647-2 Compliant Workflow

Bincai operates under ISO 12647-2:2013 (Process control for offset lithography). This international standard defines:

  • Paper type classifications (we run PT1–PT5: gloss coated to uncoated)
  • Target CIELAB values for CMYK solids
  • TVI (Tone Value Increase) curves
  • Gray balance tolerances

Every job at Bincai is profiled against ISO 12647-2 using custom ICC profiles generated with an X-Rite i1iO automated scanning table and i1Profiler software. We maintain separate profiles for:

  • Coated paper (FOGRA 51/52)
  • Uncoated kraft (custom profile)
  • Specialty boards (metallic, black, colored substrates)
  • UV-cured inks (FOGRA 53 equivalent)

The 6-Stage Color QC Process at Bincai

Stage 1: Prepress Color Proofing

Before plates are made, every job receives a GMG ColorProof digital proof on the actual production substrate, verified against a FOGRA Media Wedge under D50 (5000K) standardized lighting.

Stage 2: Plate Quality Control

Computer-to-Plate (CTP) output is checked with a Techkon SpectroPlate to ensure dot percentages match the RIP data within 1% across the entire tonal range.

Stage 3: Makeready Color OK Sheet

The press crew pulls an OK sheet, measures it against the GMG proof, and obtains customer sign-off (physical or digital via calibrated photo) before the production run begins.

Stage 4: Inline Continuous Monitoring

Heidelberg Prinect Inpress Control scans every sheet. Any sheet exceeding ΔE 2.0 triggers an automatic ink-key adjustment and a side-mark for manual inspection.

Stage 5: Periodic Offline Verification

Every 1,000 sheets, quality inspectors pull a sample and measure:

  • Solid ink density (CMYK + spot colors)
  • Dot gain at 25%, 50%, 75%
  • Gray balance patch
  • Registration marks (±0.05mm tolerance)

Stage 6: Pre-Shipment Color Audit

Before palletizing, random AQL sampling (typically Level II, AQL 1.0) includes color measurement against the OK sheet standard. Any deviation triggers a 100% re-inspection of that production lot.

PMS Spot Color Matching: Beyond CMYK

While CMYK process printing handles most photographic imagery, Pantone Matching System (PMS) spot colors are essential for:

  • Corporate logos (Coca-Cola red, Starbucks green)
  • Metallic and fluorescent colors
  • Brand-specific pastels that CMYK cannot reproduce
  • Large solid areas requiring absolute uniformity

Bincai maintains an extensive Pantone Formula Guide library (coated, uncoated, and metallic) and uses X-Rite InkFormulation Software to mix custom spot colors from our in-house ink kitchen. Our ink technicians average 15+ years of experience matching PMS colors by eye and instrument.
