Product Overview

The book-style magnetic closure photography portfolio box with archival acid-free linen wrap is purpose-built for professional photographers, fine art printers, and galleries who require museum-grade storage and presentation. Manufactured by Guangdong Bincai Color Printing (founded 2003, Foshan, 18,000 sqm, ISO 9001:2015 & FSC-C147399 certified), this box features acid-free, lignin-free, pH-buffered materials throughout — from the 3.0mm archival greyboard core to the Belgian linen wrap and pH-neutral adhesive. The book-style magnetic closure with N52 neodymium magnets opens like a fine volume, and the spine can be foil-stamped with the photographer’s name or collection title. Compatible with ISO 18902:2013 imaging materials storage standards. Daily output reaches 1.7 million boxes on KBA Rapida 105 and Heidelberg CD 102 7+1 UV presses.

Production Specifications

  • Press: Heidelberg CD 102 7+1 UV for linen-adhered label printing; KBA Rapida 105 for inserts
  • Die-Cutting: Bobst SP 102 BMA/E for precision board cutting; Bobst Visioncut 106 LER
  • Board: 3.0mm archival greyboard — pH 7.5–8.5, alkaline-buffered (3% calcium carbonate), lignin-free
  • Linen: 100% Belgian linen, acid-free, passed PAT (ISO 18916)
  • Adhesive: pH-neutral PVA, solvent-free, passed PAT
  • Magnet: N52 neodymium, machine-pressed flush into board
  • Folding/Gluing: Bobst Expertfold 110 A2 with book-style spine forming
  • Daily Output: 1.7M color boxes + 220,000+ premium boxes
